# IPFlagTray

Приложение для системного трея Windows, отслеживающее внешний IP-адрес и отображающее флаг страны.

## Возможности

- Автоматическое определение внешнего IP-адреса (ifconfig.me)
- Определение страны по IP (ipwho.is)
- Отображение флага страны в системном трее
- Уведомление при смене IP на российский
- Автообновление каждую минуту

## Требования

- .NET 8.0 SDK
- Windows

## Сборка

```bash
dotnet build
```

## Публикация

```bash
dotnet publish -c Release -r win-x64 --self-contained
```

## Использование

После запуска приложение отображается в системном трее. Меню доступно по клику правой кнопкой мыши:

- Текущий IP и страна
- Обновить сейчас
- Открыть проверку IP
- Выход

При обнаружении российского IP показывается диалоговое окно с предупреждением.
